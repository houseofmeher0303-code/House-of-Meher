# House of Meher - Farshi Salwars & Kurtis

Beautiful, minimal e-commerce storefront for premium farshi salwars and coordsets.

## 🚀 Quick Start

### Prerequisites

- Node.js installed on your computer
- GitHub account
- Vercel account

### Installation

1. Clone this repository

```bash
git clone https://github.com/YOUR-USERNAME/house-of-meher.git
cd house-of-meher
```

1. Install dependencies

```bash
npm install
```

1. Run locally

```bash
npm start
```

1. Open <http://localhost:3000>

## 📦 Features

- ✨ Minimal, elegant design
- 🛍️ Easy shopping cart
- 💳 Razorpay payment integration
- 📱 Mobile responsive
- 🎯 Left sidebar navigation
- ❤️ Wishlist functionality

## 🛠️ Setup Razorpay

1. Go to [razorpay.com](https://razorpay.com)
1. Create account and complete KYC
1. Get your API Key ID from dashboard
1. In `src/HouseOfMeher.jsx`, replace `'YOUR_RAZORPAY_KEY_ID'` with your actual key

## 📸 Adding Products

To update products with real images:

1. Upload photos to [cloudinary.com](https://cloudinary.com)
1. Get image URLs
1. Update `products` array in `src/HouseOfMeher.jsx`

## 🌐 Deploy to Vercel

1. Push code to GitHub
1. Go to [vercel.com](https://vercel.com)
1. Click “New Project” → Import repository
1. Click Deploy
1. Your website is live!
